Projects

  • the Platform for "Future" Oversea Education Agency Based on SpringBoot   2020

    The platform uses the SpringBoot framework to develop the back-end. Based on that, using incremental JavaScript framework vue.js family barrel and Bootstrap-based jqGrid to develop the front-end. With Thymeleaf as a template-driven to complete the construction of the MVVM architecture.

  • User-Identification System Based On the Gesture Operation   2019

    The project can analyze whether the person who is operating the phone at the moment is the owner of the phone by collecting user gestures. Meanwhile, the project can use the ontouchevent() method of android to collect gesture data. Python script was invoked in the back-end. The optimal parameter range was determined by grid search. The second classification of user gestures was successfully realized by using K-NearestNeighbor training model, which was manifested in the android terminal as exiting the APP or locking the screen normally.

  • Genting House   2018

    Using Maven to build the Genting House and used the SSM (Spring, SpringMVC, MyBatis) framework and AdminLTE to implement the design.

    Contribution: Using AdminLTE to write front-end. Using SSM framework to compile user's back-end logic. Relying on stored procedures in MySQL database to handle transactions. PageHelper is used to break the form up over multiple pages, so as to simplify the interface.
  • Let Big Data Tell You Whether You Want to Have a Second Child   2017

    Using Matlab to complete data analysis and prediction for the next 15 years under different population policies.

    Contribution: Based on Logistics model to analyze and predict data. Using Back Propagation Neural Networks to predict data. Using Edraw to visualize data.
  • Reconstruction of Road Traffic Flow Based on Multi-source Monitoring Data   2017
    Contribution: Using Entropy Weight Method and K-means clustering analysis to develop traffic indices model. Using Markov chain to forecast system situation in the future.
